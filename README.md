# afknot Documentation

This repository contains Markdown documentation for the afknot project.  
Built with [MkDocs](https://www.mkdocs.org/).

## Local Development

1. Install MkDocs:
   ```
   pip install mkdocs
   ```
2. Serve locally:
   ```
   mkdocs serve
   ```

3. Build for production:
   ```
   mkdocs build
   ```

4. Deploy to GitHub Pages:
   ```
   mkdocs gh-deploy
   ```
